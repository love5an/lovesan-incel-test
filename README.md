# How incel is love5an? / Насколько love5an инцел?

A bilingual (EN/RU) joke quiz. Single static `index.html`, no build, no
dependencies — just open it or host it on GitHub Pages.

🔗 **Live:** https://love5an.github.io/lovesan-incel-test/

It's parody. The "incel" framing is the self-deprecating lonely-Lisp-hacker meme
(drinks alone, argues about parentheses, `git log` older than the last date) —
not a diagnosis of anyone real. Pairs with
[bike-gram](https://github.com/love5an/bike-gram) and
[beergram](https://github.com/love5an/beergram).

## Run locally

```sh
open index.html        # macOS
# or just double-click it
```
